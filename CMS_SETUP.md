# Pages CMS

1. Carrega esta versão para o GitHub.
2. Confirma que `.pages.yml` está na raiz.
3. Abre https://app.pagescms.org/.
4. Faz login, escolhe o repositório e abre **Receitas**.

No site, a página **Adicionar** tem um botão direto para abrir o Pages CMS. O gerador JSON ficou colapsado como Plano B.
