# Livro de Receitas do Miguel

Site estático de receitas para GitHub Pages.

## Pages CMS

Esta versão inclui `.pages.yml` para editar receitas e fotos no Pages CMS. A página **Adicionar** tem link direto para `https://app.pagescms.org/`.

## Plano B

Na página **Adicionar**, o gerador de JSON continua disponível colapsado para o caso de o Pages CMS falhar.


## Fluxo recomendado de edição

Na página **Adicionar receita**, usa o botão **Abrir Pages CMS** para editar receitas e fotos. O gerador de JSON continua disponível como plano B, colapsado na mesma página.


## Pages CMS experimental

Esta versão inclui `.pages.yml` para testar edição de receitas e fotos através do Pages CMS. Consulta `CMS_SETUP.md` para instruções.

## Documentação adicional

- `CMS_SETUP.md` — instruções para testar e usar Pages CMS.
- `RELEASE_NOTES.md` — histórico de versões e alterações.
- `PROJECT_HISTORY.md` — histórico do que foi pedido e como foi feito.
- `CODE_REVIEW.md` — revisão técnica da versão final.
